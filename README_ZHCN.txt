============================================
基本操作：

鼠标左键   拖动控制视角转换
鼠标滚轮   放大缩小视角
F1/F2      增加/减少阶数
F5         从in.txt文件中读取指令
数字键1-3  不同的魔方花样
ESC        停止旋转/复原魔方
空格       随机旋转一次
回车       随机旋转100次

============================================

in.txt文件说明
程序中蓝色面为正面 黄色为上面
每一个指令由三部分组成 多个指令之间以【空格】分隔

前缀[可以没有]为偏移量

L/R     控制左面/右面旋转
U/D     控制顶面/底面旋转
F/B     控制前面/后面旋转

无后缀  顺时针旋转
后缀为' 逆时针旋转
后缀为2 旋转180度

以下是单个指令的例子

L     左面顺时针旋转
L'    左面逆时针旋转

D2    底面旋转180度
2R    从右起第二个面旋转180度

3U'   从顶面起第三个面逆时针旋转一次

以下是一些允许的in.txt例子
F B L R F B L R F B L R
U' D F' B L R' U' D
R2 F2 B2 L2 U F2 R2 L2 B2 D'
